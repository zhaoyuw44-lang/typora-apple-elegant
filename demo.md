## Typography & Inline Styles

This paragraph demonstrates the standard body text. It uses the **Inter** font family, which has been fine-tuned with a letter-spacing of `-0.011em` and a comfortable line-height of `1.625` to mirror the layout of **Apple.com**.

You can emphasize your words using **bold text** (`strong`) or *italicized text* (`em`). If you need to show keys or shortcuts, you can use the keyboard tag like <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd>, which renders as high-fidelity tactile keycaps.

## macOS-Terminal Code Blocks

Our fenced code blocks simulate a native macOS terminal window, including rounded corners, soft drop shadows, and the iconic **Red, Yellow, and Green traffic light buttons** in the top-left corner.

```javascript
// This is an Xcode-inspired syntax highlighting preview!
import { typoraTheme } from 'apple-elegant';

function initializeWorkspace(themeName) {
    const isAppleStyle = themeName === "Apple Elegant";
    if (isAppleStyle) {
        console.log("🍏 Custom macOS theme loaded successfully.");
        return {
            status: "Premium",
            active: true
        };
    }
    return null;
}

// Initialize the theme
const myTheme = initializeWorkspace("Apple Elegant");
```
